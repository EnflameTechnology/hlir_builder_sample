enum builder::PrintingFlags¶

Set of flags used to control the behavior of the Builder::Print methods.

Values:

enumerator None¶
enumerator ElideLargeElementsAttrs¶

Enable the elision of large elements attributes, by printing a ‘…’ instead of the element data.

Note: The IR generated with this option is not parsable. largeElementLimit = 16

enumerator EnableDebugInfo¶

Enable printing of debug information.

If ‘prettyForm’ is set to true, debug information is printed in a more readable ‘pretty’ form. Note: The IR generated with ‘prettyForm’ is not parsable.

enumerator EnableDebugInfoPrettyFormFlag¶
enumerator PrintGenericOpForm¶

Always print operations in the generic form.

enumerator UseLocalScope¶

Use local scope when printing the operation.

This allows for using the printer in a more localized and thread-safe setting, but may not necessarily be identical to what the IR will look like when dumping the full module.
